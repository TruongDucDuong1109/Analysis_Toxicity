
# Analysis Toxicity



## Installation

Install framework:

```bash
pip install matplotlib
pip install nltk
pip install scikit-learn

```

    
## Running Tests

Assign toxicity field to comments

```bash
  py toxicity.py
```

Analyze the level of toxicity in comments
```bash
  py analysis.py
```

Visualize Data
```bash
  py visualize_data.py
```
